# Examen-app

Este es mi fracaso menos feo de una aplicación web que crea exámenes tipo COMIPEMS. La aplicación permite capturar preguntas, crear exámenes y resolver exámenes.

## Requerimientos

- Node: 16.15.1
- Angular CLI: 14.0.2
- Angular 14.0.2
- mongodb 5.0.9

## Desarrollo

Installar paquetes

```
cd ./front
npm install
cd ../back
npm install
```

Inicializar mongodb

```
systemctl start mongod
systemctl status mongod
```

Iniciar backend

```
cd back/
npm start # o npm dev
```

Iniciar front

```
cd front/
npm start # o ng serve
```
